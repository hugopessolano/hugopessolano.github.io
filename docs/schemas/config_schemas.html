Esquemas de Configuración (config_schemas)

Defines Pydantic models for store-specific configurations, primarily language settings.

This module provides the StoreConfig model to manage which languages are enabled or should be modified during operations like i18n parsing.

schemas.config_schemas.DEFAULT_CONFIG = StoreConfig(enabled_languages=['es', 'en', 'pt'], modify_languages=['es'])

Default store configuration instance with Spanish, English, and Portuguese enabled, but modifications primarily target Spanish.

class schemas.config_schemas.StoreConfig(**data)[fuente]

Bases: BaseModel

Configuration model for store language settings.

Variables:

  • enabled_languages (List[str]) – A list of language codes (e.g., «es», «en») that are considered active or enabled for the store. Used, for example, to check if an existing i18n field structure is valid.

  • modify_languages (List[str]) – A list of language codes for which values should be generated or modified when parsing non-i18n input into an i18n structure.
