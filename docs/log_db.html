Configuración de Logging (log_db)

Configures logging using Loguru, directing logs to both standard output and an SQLite database file.

Provides a custom SQLite sink (SQLiteSink) to handle database writing and configures multiple logger instances (stdout_logger, db_logger, db_only_logger) with different filtering rules to control where log messages are sent based on their intended destination (stdout only, stdout and DB, DB only).

log_db.DB_FILE = 'execution_logs.db'

The name of the SQLite database file for storing logs.

class log_db.SQLiteSink(db_path)[fuente]

Bases: object

A Loguru sink class that writes log records to an SQLite database.

Handles database connection, table creation, writing formatted log records, and closing the connection gracefully on program exit.

Variables:

  • db_path (str) – Path to the SQLite database file.

  • conn (sqlite3.Connection | None) – The active SQLite connection object.

  • cursor (sqlite3.Cursor | None) – The database cursor for executing SQL.

Parámetros:

db_path (str)
