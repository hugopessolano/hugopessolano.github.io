Utilidades de DataFrame (dataframe_utils)

Utility functions for processing and validating DataFrame data, particularly for internationalization (i18n) fields and payload reformatting based on Pydantic models.

dataframe_utils.is_field_type_i18n(model, field_name)[fuente]

Checks if the type annotation of a field in a Pydantic model is i18nField or Optional[i18nField].

Parámetros:

  • model (Type[ValidationModel]) – The Pydantic model class.

  • field_name (str) – The name of the field to check within the model.

Devuelve:

True if the field type is i18nField or Optional[i18nField],

False otherwise.

Tipo del valor devuelto:

bool

dataframe_utils.parse_i18n(payload_arguments, key, store_config)[fuente]

Parses or transforms a value in the payload dictionary for an i18n field.

If the value is a valid JSON string representing an i18n structure (dict with language codes as keys), it’s parsed and used directly. Otherwise, the value is replicated for all languages specified in store_config.modify_languages. Handles potential NaN values from pandas.

Parámetros:

  • payload_arguments (Dict[str, Any]) – The dictionary containing payload data. This dictionary is modified in-place.

  • key (str) – The key in payload_arguments corresponding to the i18n field.

  • store_config (StoreConfig) – The store configuration containing language settings.

Devuelve:

The payload_arguments dictionary is modified directly.

Tipo del valor devuelto:

None

dataframe_utils.reformat_payload(payload_arguments, keys_to_reformat, validation_model, store_config=StoreConfig(enabled_languages=['es', 'en', 'pt'], modify_languages=['es']))[fuente]

Reformats payload arguments based on the expected validation model and store configuration.

Specifically handles i18n fields by calling parse_i18n and aggregates “option_value_X” fields into a “values” list for the Variant model.

Parámetros:

  • payload_arguments (Dict[str, Any]) – The dictionary of arguments extracted from a data row, intended as payload. This dictionary might be modified.

  • keys_to_reformat (List[str]) – A list of keys within payload_arguments that potentially need reformatting based on the validation_model.

  • validation_model (Type[ValidationModel]) – The Pydantic model used to validate and determine the structure of the payload.

  • store_config (StoreConfig, optional) – The store configuration, primarily for language settings. Defaults to DEFAULT_CONFIG.

Devuelve:

The potentially modified payload_arguments dictionary

with fields reformatted according to the model and configuration.

Tipo del valor devuelto:

Dict[str, Any]

Muestra:

ValueError – If the payload contains both a “values” key and “option_value_X” keys when the validation model is Variant.
