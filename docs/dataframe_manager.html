Gestor de DataFrames (dataframe_manager)

Manages pandas DataFrames, particularly for preparing data for API requests.

This module provides the DataframeManager class, which facilitates reading data from CSV files or using existing DataFrames, converting rows into structured request objects, and handling potential errors during the process.

class dataframe_manager.DataframeManager(csv_name=None, df=None)[fuente]

Bases: object

Handles operations on a pandas DataFrame, primarily focused on converting rows into API request objects based on endpoint configurations.

Can be initialized either by reading a CSV file from the “tmp” directory relative to the script location or by directly passing a pandas DataFrame.

Variables:

  • _id (uuid.UUID) – A unique identifier for the manager instance.

  • _csv_name (str | None) – The name of the CSV file used for initialization, if any.

  • _script_dir (str) – The directory where the script is located.

  • _file_path (str | None) – The full path to the input CSV file, if used.

  • _export_file_path (str) – The full path for exporting data.

  • _current_df (pd.DataFrame) – The pandas DataFrame being managed.

Parámetros:

  • csv_name (str | None)

  • df (DataFrame | None)

parse_requests_into_dataframe(request_type, endpoint)[fuente]

Applies row_to_request to each row of the internal DataFrame.

Adds a new column named “request” to the DataFrame, containing the generated Request object for each row (or None if conversion failed). Logs statistics about successful and failed conversions.

Parámetros:

  • request_type (type[Request]) – The specific Request class to use for conversion.

  • endpoint (Endpoint) – The Endpoint configuration for the requests.

Devuelve:

The DataFrame with the added “request” column.

Tipo del valor devuelto:

pd.DataFrame

Muestra:

ValueError – If the DataFrame is not empty but no requests could be successfully parsed.

row_to_request(row, request_type, endpoint)[fuente]

Converts a single DataFrame row into a specified Request object.

Dynamically maps row data to the Request model fields based on the provided Endpoint configuration. Handles positional URL arguments, optional “params”, and constructs the “payload”.

Parámetros:

  • row (pd.Series) – The DataFrame row to convert.

  • request_type (type[Request]) – The specific Request class (e.g., GetRequest, PostRequest) to instantiate.

  • endpoint (Endpoint) – The Endpoint configuration object containing details about path, arguments, and validation models.

Devuelve:

An instance of the specified “request_type” populated with data from the row, or None if conversion fails.

Tipo del valor devuelto:

Request | None
